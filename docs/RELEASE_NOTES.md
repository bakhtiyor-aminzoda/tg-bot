# Release Notes — UX Boost

Последние улучшения Telegram Video Downloader Bot, сфокусированные на пользовательском опыте.

## Highlights
- **Локализация по умолчанию.** Бот автоматически подстраивается под язык Telegram-клиента (RU/EN), включая статусы, кнопки и подписи.
- **Живые статусы прогресса.** Промежуточные обновления показывают %, скорость, ETA и уже загруженный объём без спама.
- **Авто-режим без клавиатур.** В личных чатах загрузка сразу стартует в лучшем доступном качестве — меньше шагов и меньше ошибок при запуске.
- **Группы.** Сообщения со ссылками получают кнопку «⬇️ Скачать/Download», которая запускает скачивание в том же чатике и показывает статус всем участникам.
- **Релизный чек-лист.** Добавлен `docs/QA.md` — чек-лист ручного прогона сценариев (личные чаты, группы, ошибки, rate-limit).
- **Новые тесты.**
  - `tests/test_handlers_downloads.py` — покрывает cooldown, групповые токены и send_document fallback.
  - Логи во время тестов приглушены (нет шумных yt-dlp сообщений).

## Suggested Talking Points (Telegram / Release Post)
1. «Теперь бот показывает живой прогресс скачивания (скорость, ETA, объём) на вашем языке.»
2. «В личке бот сразу стартует скачивание в авто-режиме — никаких лишних тапов.»
3. «Группы получили кнопку “Download”, поэтому ссылку видят все, а скачивание стартует после нажатия.»
4. «Подготовили чек-лист `docs/QA.md`, чтобы QA-команда быстро проверяла новый UX перед релизом.»
